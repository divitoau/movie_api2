Super Cool Rad Movie Info

This API provides users with access to information about different movies, directors, and genres.

Business Logic URL HTTP Method Request body data format Response body data format
Get list of all movies /movies GET none A JSON object holding data
about all movies
Get movie data by title /movies/[Title] GET none A JSON object holding data
about a single movie
structured like
{
"title" : "Sucker Punch",
"description" : "[description]",
"genre" : "Action",
"director" : "Zack Snyder",
"imageURL" : "[image URL]",
"isFeatured" : "false"
}
Get genre description by name /movies/[Title]/[Genre] GET none A text description
Get director data by name /movies/[Title]/[Director] GET none A JSON object structured like:
{
"name" : "Austin DiVito",
"bio" : "[bio]",
"birthYear" : "1996",
"deathYear" : "2095"
}
Register new user /users POST A JSON object structured like:
{
"name" : "Austin DiVito",
"username" : "CoolDude475"
} 		A JSON object structured like:
{
"id" : "uvwxyz",
"name" : "Austin DiVito",
"username" : "CoolDude475"
}
Update username /users/[ID]/[username] PUT none A text message confirming the
new username
Add movie to favorites list /users/[ID]/favorites/[Title] POST none A text message confirming the
movie that was added
Remove movie from favorites list /users/[ID]/favorites/[Title] DELETE none A text message confirming the
movie that was removed
Deregister user /users/[ID] DELETE none A text message confirming the
user was deregistered
